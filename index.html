Course Description:

This course will provide a comprehensive study of programming syntax, algorithms and data structures. Students will be expected to study and master a sufficient amount of material to perform well on the AP Computer Science A exam.

The content will include, but not be limited to, the following topics:

  • Java language syntax
  • Defining variables and arithmetic expressions
  • Introduction to classes and object-oriented programming
  • Conditionals and looping
  • The String class
  • The ArrayList class
  • Creating, declaring and initializing arrays
  • Searching and sorting arrays
  • Classes, inheritance and interfaces
  • Recursion
